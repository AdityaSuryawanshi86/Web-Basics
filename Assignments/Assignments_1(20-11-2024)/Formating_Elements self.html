The 'b tag' in HTML is used to make text bold, typically for emphasis or to highlight certain content.

The 'i tag' in HTML is used to italicize text, often to indicate a change in mood, technical terms, or foreign words. with a bat and ball.

The 'sup tag' in HTML is used to display text as superscript, typically for mathematical exponents or footnotes. eg. TeamA

The 'sub tag' in HTML is used to display text as subscript, typically for chemical formulas or mathematical notation. eg. TeamB

The 'small Tag' - It is used in HTML to display text in a smaller font size, typically to indicate fine print, disclaimers, or secondary information.

The 'big tag' in HTML is used to display text in a larger font size, typically for emphasis or to highlight certain content.
(Note: The big tag is now obsolete in HTML5, and it's recommended to use CSS for font size adjustments.)

The 'strike tag' in HTML is used to display text with a line through it, indicating that the content is no longer relevant or has been deleted.
(Note: The strike tag is obsolete in HTML5; del or s are recommended alternatives.)

The 'span tag' in HTML is an inline container used to group and style a portion of text or elements without affecting the layout. - Span

The 'q tag' in HTML is used to define a short inline quotation, typically displaying text within quotation marks.

The 'mark tag' in HTML is used to highlight or emphasize text, typically indicating a search term or important content.

The 'hr tag' in HTML is used to create a horizontal rule or line, typically used to separate content or sections within a page.

The 'h1 tag' in HTML is used to define the main heading of a page or section, typically representing the most important content.



The 'h2 tag' in HTML is used to define a second-level heading, typically representing a subsection or subheading in the content, and is typically displayed in a larger and bolder font.
